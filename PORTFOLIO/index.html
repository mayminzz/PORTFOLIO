웹 개발에 관심이 많아 열심히 공부중입니다.

머릿속 아이디어를 눈앞에 구현하는 것에 매력을 느낍니다.

포트폴리오에 방문해 주셔서 감사합니다!🙇‍♀️

Tech Stack

Design 디자인

  • Photoshop, Illustrator를 이용한 디자인 및 편집 작업이 가능합니다.
  • Figma를 이용한 UI/UX 설계와 프로토 타이핑이 가능합니다.

FrontEnd 프론트엔드

  • 웹 표준, 웹 접근성 마크업 작업이 가능합니다.
  • 반응형, 모바일 웹에 대한 이해 및 제작이 가능합니다.
  • HTML, CSS, SCSS를 이용한 퍼블리싱이가능합니다.
  • Javascript, jQuery를 이용한 동적인 웹 구현이 가능합니다.
  • React 프레임 워크와 Typescript에 대한 기초 지식이 있습니다.

DataBase 데이터베이스

  • Firebase를 통한 호스팅이 가능합니다.
  • Github사용이 가능하며, 모든 작업물을 업로드하여 버전을 관리하였습니다.

Tool 툴

  • Visual Studio Code를 활용할 수 있습니다.

    Main Skill.

    타임라인 부분에서 왼쪽 책갈피 포스트잇을 눌렀을 때 슬라이드가 되면서 해당 페이지로 이동할 수 있도록 자바스크립트로 구현하였습니다.
